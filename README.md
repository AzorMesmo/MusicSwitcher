# Music Switcher
This program switch name of music and artist of *mp3* music file.

After download the *.py* file you have to create a folder named "music" on the same directory and place the musics there.
The songs files must have the name exactly as follows: "*name of artist - name of song*". The program will switch this names, for example:

- SuicideBoys - 2 Hot 4 You *to* 2 Hot 4 You - SuicideBoys
- BeamerBoy - Lil Peep *to* Lil Peep - BeamerBoy
- Post Malone - White Iverson *to* White Iverson - Post Malone
- Butterfly Effect - Travis Scott *to* Travis Scott - Butterfly Effect
